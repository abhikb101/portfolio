# PORTA Launch Website

The official launch website for PORTA - your private passport to a more trusted and accessible Superchain.

## Overview

This website implements the strategic blueprint for PORTA's launch, featuring a vision-first approach that leads with the promise of unlocking the on-chain economy. The design follows a comprehensive design system with carefully crafted typography, colors, and components.

## Design System

### Colors

- **Primary Action**: #0052FF (Electric Blue)
- **Hero & Accent**: #E74C3C (Logo Red)
- **Highlight**: #FBF0B2 (Logo Yellow)
- **Base Black**: #1A1A1A
- **Base White**: #FFFFFF
- **Subtle Gray**: #F0F0F0

### Typography

- **Display Font**: Montserrat (ExtraBold, Bold, Medium)
- **Body Font**: Inter (Regular, Medium)

### Components

- **Primary Button**: Solid blue background with hover states
- **Secondary Button**: Black border with hover fill
- **Cards**: Clean white/gray backgrounds with optional borders
- **Input Fields**: Subtle gray borders with blue focus states

## Website Structure

1. **Hero Section**: Bold red background with the main vision statement
2. **The Problem**: Clean white section explaining the current challenges
3. **The Solution**: Gray background showcasing the Human Score concept
4. **The Future**: Dark section describing the passport evolution
5. **For Protocols**: Developer-focused integration section
6. **Final CTA**: Red background with waitlist signup form

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technology Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **Google Fonts** - Montserrat and Inter fonts

## Features

- ✅ Responsive design for all screen sizes
- ✅ Smooth scrolling navigation
- ✅ Interactive components with hover states
- ✅ Email waitlist form
- ✅ Clean, modern design following the blueprint
- ✅ Accessibility considerations
- ✅ Performance optimized

## Deployment

The website is ready for deployment on Vercel, Netlify, or any other Next.js-compatible platform.

```bash
npm run build
npm start
```
